Priori DB

This project was built in my free time out of passion. It's a database management tool designed to be as practical and easy to use as possible while offering a clean and intuitive interface. With it, you can fully manage databases (MySQL and PostgreSQL), execute queries, manage data, metadatas, internal params, and everything else. Thanks to anyone interested! üòÅ - Leonardo Priori

Advanced Query Editor and Executor

Powerful and complete SQL editor with syntax highlighting, queries history, favorite queries, rich data grid, incremental fetching, STDIN and STDOUT PostgreSQL support, tab renaming, and more.

Powerfull Tree-Style Navigation

Effortlessly browse through your database structure with a intuitive tree navigation, with easy search capbalities, easy access to adjustments, table datas, metadatas, functions, domains, and more.

Interactive Data Grid

Powerful data grid for database records with editing, filtering, incremental fetching, sorting, etc.

A big list of features

You can modify any internal configuration of the database, parameters, manage users, roles, permissions, sequences, funcions, triggers, indexes, comments, and more.
